# AImBTI — Product Requirements Document

## 개요
AI 생존력 진단 서비스 AImBTI의 제품 요구사항 문서입니다.
20가지 질문으로 사용자의 AI 시대 생존 유형을 진단하고, 결과를 공유하며, 메타코드 서비스로 유입을 유도합니다.

---

## 1차 피드백 (2026-04-06)

### 1. 인스타그램 공유
**목표:** 결과 페이지에서 인스타그램으로 빠르게 공유할 수 있도록

**요구사항:**
- 결과 공유 섹션에 인스타그램 공유 버튼 추가
- 모바일: `navigator.share({ files: [imageFile] })` → 네이티브 공유 시트 (인스타 포함)
- 데스크탑: 이미지 다운로드 (긴 세로형 카드)
- 인스타 공유용 이미지: 세로형 (800×1400px), 내용 풍부하게

**구현 파일:** `src/app/result/[id]/ResultClient.tsx`

---

### 2. 공유 섹션 UX 전면 개선
**목표:** 공유 버튼이 더 눈에 띄고, 사용자가 자연스럽게 공유하도록

**요구사항:**
- 공유 섹션을 결과 페이지 상단으로 위치 조정 고려
- 버튼 크기 확대, 강조색 적용
- 섹션 헤드카피: "친구들한테 공유하고 놀래켜줘! 😏"
- 공유 버튼 구성: [📷 인스타 공유] [💛 카카오] [🔗 링크복사]

---

### 3. CTA 2개로 단순화

**제거:**
- ~~무료 쿠폰~~ (쿠폰 섹션 전체 삭제)
- ~~무료 상담 받기~~ (부트캠프 상담 모달 삭제)
- ~~AI 상담~~ 미포함

**유지/신규:**

#### 3-1. 무료 전자책
- 미리보기: 3~4페이지 blur 처리된 프리뷰 이미지 표시
- CTA: "메타코드 회원가입하고 무료 다운로드 →"
- 클릭 시 `NEXT_PUBLIC_METACODE_SIGNUP_URL`로 이동 (utm_source=aimbti 파라미터 포함)
- 전자책 종류: 커리어 전자책 / 부트캠프 과정소개서
- **에셋 필요:** 전자책 미리보기 이미지 (담당자 제공 필요)
- **URL 필요:** 메타코드 회원가입 URL (`NEXT_PUBLIC_METACODE_SIGNUP_URL`)

#### 3-2. 무료 특강 단톡방 (AImBTI 전용)
- 카피: "🎓 AImBTI 전용 무료 특강 단톡방 입장하기"
- 이 사이트를 통해 들어온 사람임을 식별 가능한 별도 단톡방
- **URL 필요:** `NEXT_PUBLIC_OPENCHAT_SURVEY_URL` (기존 오픈채팅과 별도)

### 4. 질문 진행 중 독려 메시지 팝업
**목표:** Q5, Q10, Q15 완료 시 사용자에게 동기 부여

**요구사항:**
- Q5 완료: "🔥 벌써 25%! 조금만 더 가보자!"
- Q10 완료: "💪 절반 달성! 이제 반 왔어요!"
- Q15 완료: "🚀 거의 다 왔어요! 5개만 더!"
- 오버레이 팝업 형식, 2초 후 자동 닫힘
- 슬라이드인 + 페이드아웃 애니메이션

**구현 파일:** `src/app/test/page.tsx`

---

### 5. 결과 등장 애니메이션
**목표:** 결과 진입 시 임팩트 있는 연출

**요구사항:**
- Confetti(폭죽) 효과: 결과 페이지 진입 시 1.5초간 타입 컬러 기반 confetti
- AI 대체 점수 게이지: 0%에서 실제값까지 1.2초 easeOutCubic 애니메이션
- 점수 숫자도 카운트업 (0 → 실제값)

**라이브러리:** `canvas-confetti`
**구현 파일:** `src/app/result/[id]/ResultClient.tsx`

---

### 6. 유형 이름 Fun하게 변경 (미구현 — 확정 필요)
**목표:** 딱딱한 유형명을 재밌고 AI 직군 연상 이름으로

**방향:** DA / AI / DE / LLM 직군 컨셉 + 재치있는 한국어 네이밍
**현재 예시:**
- HSCP: "나만의 세계 완성형" → "혼자서 우주 만드는 DA형 🌌"
- HALF: "AI 시대 지휘관" → "AI 부려먹기 최강자 👑"

**16개 전체 네이밍 확정 필요**

---

### 7. 결과 이미지 개선 (미구현 — 에셋 필요)
**목표:** 캐릭터 이미지를 더 fun하고 귀엽게

**방향:**
- AI 연상 캐릭터 이미지 교체
- 현재 SVG에 CSS 애니메이션 강화 (bounce, wiggle, sparkle)
- 또는 Lottie 애니메이션으로 교체 (에셋 제작 필요)

**에셋 필요:** 16개 타입별 새 캐릭터 이미지
