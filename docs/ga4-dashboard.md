# GA4 탐색 분석 설계

## 추적 이벤트 목록

| 이벤트 | 시점 | 파라미터 |
|--------|------|----------|
| `page_view` | 페이지 진입 시 (자동) | `page_location` |
| `cta_click` | 랜딩 CTA 버튼 클릭 | `location` |
| `test_start` | 1번 문항 답변 시 | - |
| `test_complete` | 마지막 문항 답변 후 | `type_code`, `ai_score` |
| `result_view` | 결과 페이지 진입 시 | `type_code`, `ai_score` |
| `share_click` | 공유 버튼 클릭 | `method` (kakao/link/card) |
| `coupon_copy` | 쿠폰 복사 클릭 | - |
| `bootcamp_click` | 무료 커리큘럼 버튼 클릭 | `bootcamp`, `type_code` |

### cta_click location 값

| 값 | 위치 |
|----|------|
| `header` | 헤더 우상단 버튼 |
| `hero` | 히어로 메인 CTA |
| `type_preview` | 유형 카드 하단 버튼 |
| `viral` | 바이럴 섹션 버튼 |

## 분석 1. 퍼널 탐색 — 어디서 빠지나

**목적:** 랜딩 방문부터 부트캠프 클릭까지 단계별 전환율 파악

### GA4 설정

탐색 → 깔때기 탐색 → 새로 만들기

| 단계 | 이벤트 조건 |
|------|------------|
| 1. 랜딩 방문 | `page_view` + `page_location` contains `/` |
| 2. CTA 클릭 | `cta_click` |
| 3. 테스트 시작 | `test_start` |
| 4. 테스트 완료 | `test_complete` |
| 5. 결과 조회 | `result_view` |
| 6. 부트캠프 클릭 | `bootcamp_click` |

### 해석 기준

| 이탈 구간 | 의미 | 개선 방향 |
|-----------|------|-----------|
| 랜딩 → CTA | 카피/디자인 설득력 부족 | 헤드라인, 버튼 문구 A/B 테스트 |
| CTA → 완료 | 문항 중 이탈 | 문항 수 줄이기 or 진행률 UI 개선 |
| 완료 → 부트캠프 | 결과 페이지 설득력 부족 | 부트캠프 카피, urgency 강화 |

## 분석 2. 자유 형식 — 어떤 CTA가 효과적인가

**목적:** 4개 버튼 중 어디서 클릭이 집중되는지 파악

### GA4 설정

탐색 → 자유 형식

- **행(측정기준):** 이벤트 파라미터 `location`
- **값(측정항목):** 이벤트 수
- **필터:** 이벤트 이름 = `cta_click`

### 예상 패턴 및 액션

| location | 예상 순위 | 높으면 | 낮으면 |
|----------|-----------|--------|--------|
| `hero` | 1위 | 정상 | 히어로 섹션 개선 필요 |
| `type_preview` | 2위 | 유형 카드를 상단으로 이동 고려 | - |
| `viral` | 3위 | - | 카피 교체 |
| `header` | 4위 | 정상 | - |

## 분석 3. 자유 형식 — 어떤 유형이 부트캠프로 전환되나

**목적:** 16개 유형별 부트캠프 클릭률 비교 → 결과 페이지 개선 우선순위 결정

### GA4 설정

탐색 → 자유 형식

- **행(측정기준):** 이벤트 파라미터 `type_code`
- **값(측정항목):** 이벤트 수
- **필터:** 이벤트 이름 = `bootcamp_click`

### 해석 및 액션

- 클릭률 높은 유형 → 해당 유형에 특화된 커리큘럼 강조 소재 제작
- 클릭률 낮은 유형 → 결과 페이지 `bootcampReason` 카피 개선 우선순위 지정

---

## 분석 4. 자유 형식 — 바이럴 경로 분석

**목적:** 공유 방법별 빈도 파악 → 재방문 유입 경로 확인

### GA4 설정

탐색 → 자유 형식

- **행(측정기준):** 이벤트 파라미터 `method`
- **값(측정항목):** 이벤트 수
- **필터:** 이벤트 이름 = `share_click`

### 해석 및 액션

| method | 의미 |
|--------|------|
| `kakao` 비율 높음 | 카카오 공유 UX 유지, 상단 배치 유지 |
| `link` 비율 높음 | SNS 바이럴 가능성, 오픈그래프 이미지 퀄리티 확인 |
| `card` 비율 높음 | 인스타그램 등 이미지 공유 채널 집중 |

---

## 분석 우선순위

```
1순위: 퍼널 탐색        → 전체 흐름의 병목 파악 (가장 먼저)
2순위: 유형별 부트캠프   → 수익화 전환에 직결
3순위: CTA 버튼 분석    → 랜딩 페이지 최적화
4순위: 바이럴 경로      → 데이터 충분히 쌓인 후 (1~2주 후)
```
