# 인스타그램 광고 카피 — 사무실 오피스편 (직장인 타겟)

- 플랫폼: Instagram (피드/스토리/릴스)
- 타겟: 25~40대 직장인, 이직 준비생
- 이미지: public/Adv/사무실(남자).png, 사무실(여자).png
- 링크: mcodegc.com/go/mi (인스타 추적용)
- 목표: 클릭 → 테스트 진입 → 결과 → 전자책/오픈채팅 전환

## 의견

인스타 광고로 괜찮지만 주의할 점:
- 인스타는 이미지가 90%. 카피는 짧을수록 좋음 (이미지 위 텍스트가 핵심)
- 현재 이미지(사무실+로봇)는 임팩트 있음. "뒤를 돌아보세요" "옆자리가 AI로 바뀐다면?" 같은 한 줄이 이미지 위에 들어가면 효과적
- 본문 카피는 스크롤해야 보이니까, 핵심은 이미지에 넣고 본문은 CTA 위주로
- 스토리/릴스 광고가 피드보다 CPC 낮고 도달 넓음 → 이미지를 9:16 비율로도 준비하면 좋음

💡 AI 역량 점수 확인하기 → mcodegc.com/go/mi

⚡ AI 역량 점수 무료 진단 → mcodegc.com/go/mi
