## 메인 주제 방향 — 3개 버전

---

### 버전 A — "AI 대체 가능성 진단"
> 공포/위기감 → 자기인식

| 구분 | 내용 |
|---|---|
| **메인 주제** | 직업 AI 대체 가능성 진단 |
| **서브 주제** | 나의 AI 유형 × 대체 위험도 |
| **핵심 훅** | "직장인 10명 중 6명, 자기 대체 가능성 모른 채 출근 중" |
| **CTA** | 나의 AI 대체 가능성 확인하기 (무료) |
| **소요시간** | 약 1분 |
| **결과 프레임** | 대체 가능성 % + 나의 AI 유형 |

**톤**: 긴박함 → 자기인식 → 행동

---

### 버전 B — "AI 유형 진단"
> 재미/정체성 → 커리어 방향

| 구분 | 내용 |
|---|---|
| **메인 주제** | 16가지 AI 유형 진단 |
| **서브 주제** | 당신은 어느 AI 유형? |
| **핵심 훅** | "나의 MBTI처럼, AI 시대 나의 유형은?" |
| **CTA** | 나의 AI 유형 확인하기 |
| **소요시간** | 약 1분 |
| **결과 프레임** | AI 유형명 + 유형별 특성 + 대체 가능성 |

**톤**: 가볍고 공유하기 좋음, MBTI 감성

---

### 버전 C — "AI 생존가능성 진단"
> 생존 → 전략 → 행동 (현재와 가장 가까움)

| 구분 | 내용 |
|---|---|
| **메인 주제** | 나의 AI 생존가능성 진단 |
| **서브 주제** | 16가지 AI 유형으로 보는 생존 전략 |
| **핵심 훅** | "AI 시대, 나는 살아남을 수 있을까?" |
| **CTA** | 나의 AI 생존가능성 확인하기 |
| **소요시간** | 약 1분 |
| **결과 프레임** | 생존가능성 % + 유형 + 전략 |

**톤**: 진지함 + 동기부여

### 결과 페이지 — 섹션 순서 재설계

```
1. 유형명 + 생존가능성 % (핵심 요약, 상단 노출)
2. 능력치 바 DNA (간격 확보 — 현재보다 여백 증가)
3. 강점 / 위기 / 탈출구 카드
4. AI 대체 가능성 % (직업 언급 최소화)
   ─────────────────────────────────────
5. 무료 특강 단톡방 CTA
   - 문구: "AI MBTI 전용 커리어 무료 특강"
   - 서브: "매주 글로벌 / 대기업 현직자 무료특강"
   - 버튼: 단톡방 입장 링크
   - 부트캠프 직접 언급 제거
6. 무료 전자책 다운받기
   (상세 설계 아래 참조)
7. 공유하기
   - 카카오 / 링크복사 / 인스타
```

### 전자책 섹션 상세 설계

| 항목 | 현재 | 변경 |
|---|---|---|
| 섹션 타이틀 | "무료 전자책 미리보기" | "무료 전자책 다운받기" |
| 가격 문구 | "9만원 상당 · 회원가입 시 전체 무료" | **전체 제거** |
| 이미지 레이아웃 | 세로 스크롤 | 좌우 슬라이드 (목차 7개 노출) |
| 잠금 오버레이 | 3~4페이지 잠금 + 회원가입 유도 | 제거 → 7개 이상 클릭 시 메타코드 사이트 팝업 |
| 이미지 비율 | 세로 길게 | 정사각형 (썸네일) |
| 저장용 이미지 | - | 페이지 전체 길이 버전 별도 |
| 버튼 문구 | "무료로 받기" | 이벤트 형식 (예: "지금 무료로 받기 →") |
| 회원가입 관련 문구 | 여러 곳 노출 | **전체 제거** |

**목차 슬라이더 동작**
- 좌우 화살표로 목차 아이템 탐색
- 한 번에 최대 7개 노출
- 8번째 이상 클릭 or 전체보기 → 메타코드 사이트로 이동 (팝업)

## 결정 필요 사항

1. **버전 선택**: A / B / C 또는 A+B 조합
   - A+B 조합 예시: 메인 훅은 "대체 가능성" (A), 결과 프레임은 "AI 유형" (B)
2. 전자책 잠금 완전 제거 vs. 메타코드 팝업 전환 방식 확정
3. 단톡방 링크 환경변수 (`NEXT_PUBLIC_OPENCHAT_URL`) 확인
