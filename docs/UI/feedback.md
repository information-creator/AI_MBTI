# UI/UX 피드백 — 데이터 기반 분석

## 퍼널 현황

```
페이지 방문 → 시작 버튼 클릭 (약 -40% 이탈)
→ 첫 문항 응답 → 20문항 완료 → 결과 확인
→ 전자책 클릭 21.6% | 오픈채팅 클릭 11.0% | 공유 클릭
```

---

## 1. 랜딩 페이지 (page.tsx) — 이탈 40%의 원인

### 문제: 히어로에서 "뭘 얻는지" 안 보임
- 현재 카피: "AI가 당신 월급을 노리고 있습니다" → 위기감만 있고 행동 동기 없음
- 서브카피: "16가지 AI 생존유형" 한 줄 → 너무 얇음, 구체적 결과물이 안 보임
- 사용자 생각: "그래서 이거 하면 뭐가 나오는데?" → 답 없음 → 이탈

### 문제: 통계 배지가 CTA 아래에 있음
- "16가지 / 20문항 / 무료" 배지가 CTA 버튼 아래
- 신뢰 요소를 보기 전에 버튼을 눌러야 하는 구조
- 순서가 반대 → 배지를 CTA 위로 올려야 함

### 문제: 상단 경고 배너 역효과
- "⚠️ 직장인 10명 중 8명 AI 대체 위험" → 광고/스팸 느낌
- 인스타 광고 타고 들어온 사용자는 이미 위기감이 있는 상태
- 랜딩에서 또 겁주면 "뭐야 이거" 하고 나감
- 제거하거나, "OO명이 진단 완료" 같은 사회적 증거로 교체 권장

### 문제: 광고 카피 vs 랜딩 카피 톤 불일치
- 광고: "나의 AI 대체율 확인" "커리어 방향까지"
- 랜딩: "월급을 노린다" → 톤이 다름
- 광고에서 기대한 것(대체율 진단)과 랜딩에서 보여주는 것(공포)이 불일치

### 문제: 유형 미리보기가 너무 아래
- 16개 카드가 스크롤 한참 아래에 있음
- 40% 이탈이면 대부분 히어로만 보고 나감
- 유형 카드 2~3개를 히어로 바로 밑에 티저로 보여주면 호기심 유발 가능

### 개선 방향
- 서브카피 강화: "나의 AI 대체율 + 살아남는 커리어 방향까지 알려드립니다"
- 통계 배지 → CTA 위로 이동
- 경고 배너 → 사회적 증거로 교체 ("지금까지 N명 진단 완료")
- 유형 카드 티저 2~3개를 히어로 섹션에 배치

## 2. 테스트 페이지 (test/page.tsx) — 양호

### 잘 된 점
- 프로그레스바 + 문항 번호 표시로 진행감 있음
- 파트별 컬러/아이콘 구분으로 지루함 방지
- 독려 토스트 (25%/50%/75%) → 이탈 방지에 효과적
- 하단 파트 인디케이터로 전체 진행 상황 한눈에 보임
- 이전 버튼으로 수정 가능 → 사용자 통제감

### 확인 필요
- 파트별 이탈률 (test_progress 이벤트로 확인 가능)
- 특정 문항에서 이탈이 집중되는지 Clarity 세션으로 체크

---

## 3. 분석 중 페이지 (analyzing/page.tsx) — 양호

### 잘 된 점
- 프로그레스바 + 단계별 라벨 ("답변 수집 중 → AI 패턴 분석 → 유형 계산 → 결과 생성")
- DB 저장과 프로그레스바 병렬 실행 → 체감 로딩 빠름
- DB 실패 시 local 폴백 → 결과를 못 보는 상황 없음

### 미세 개선 가능
- 현재 로봇 이모지(🤖)만 있음 → 유형 관련 힌트를 살짝 보여주면 기대감 상승
- "당신의 유형이 곧 공개됩니다" 같은 카피 추가 가능

## 4. 결과 페이지 (ResultClient.tsx) — 개선 여지 있음

### 잘 된 점
- 폭죽 효과 + 카운트업 애니메이션 → 임팩트 있음
- 레이더 차트로 5차원 시각화 → 이해하기 쉬움
- 인사이트 3단 구조 (강점/위기/방향) → 핵심 메시지 전달 좋음
- 공유 카드 자동 생성 (카카오/인스타/이미지) → 바이럴 구조 갖춤

### 문제: CTA 섹션 순서
- 현재 순서: 결과카드 → AI대체점수 → 직무추천 → 전자책 → 오픈채팅 → 공유 → 다시테스트
- 전자책 전환율(21.6%)이 오픈채팅(11.0%)보다 2배 높음 → 순서 맞음 (이미 수정됨)
- 그러나 공유 섹션이 전자책/오픈채팅 뒤에 있어서 바이럴 전환이 낮을 수 있음

### 문제: 전자책 섹션 CTA가 약함
- "무료 전자책 다운받기" → 50페이지 분량이라는 건 버튼에만 있음
- 전자책 가치를 더 강조하면 21.6% → 더 올릴 수 있음
- "AI 시대 생존 로드맵 6개월 플랜 포함" 같은 구체적 가치 어필

### 문제: 오픈채팅 섹션 전환율 낮음 (11%)
- "AI·데이터 무료 특강 / 매주 대기업·현직자 노하우" → 너무 일반적
- 유형별로 다른 메시지를 주면 전환율 상승 가능
- 예: "DE 전략가 유형에게 추천하는 데이터 분석 무료 특강"

### 문제: 결과 설명 텍스트가 길다
- description이 4~5줄 → 모바일에서 스크롤 많아짐
- 핵심 메시지는 인사이트 3단(강점/위기/방향)에 이미 있음
- description을 2~3줄로 줄이거나, 접기 처리 고려

## 5. 전체 퍼널 관점

### 트래킹 추가 권장
- `exit_click` 이벤트가 GA에만 있고 Supabase에는 없음 → 대시보드에서 이탈 경로 분석 불가
- 공유 방법별(카카오/인스타/이미지/링크) 전환율을 대시보드에 추가하면 바이럴 채널 최적화 가능

### 광고 → 랜딩 일관성
- 광고 카피가 "대체율 + 커리어 방향"을 약속하는데 랜딩이 "월급을 노린다"면 불일치
- 광고별로 랜딩 카피를 맞추거나, 랜딩을 광고 톤에 맞게 통일

## 우선순위 정리

| 순위 | 작업 | 예상 효과 | 난이도 |
|---|---|---|---|
| 1 | 히어로 서브카피 강화 ("대체율 + 방향까지") | 이탈 40% → 25~30% | 낮음 |
| 2 | 통계 배지를 CTA 위로 이동 | 버튼 클릭률 상승 | 낮음 |
| 3 | 상단 경고 배너 → 사회적 증거로 교체 | 첫인상 개선 | 낮음 |
| 4 | 오픈채팅 CTA를 유형별 맞춤 메시지로 변경 | 11% → 15%+ | 중간 |
| 5 | 전자책 가치 어필 강화 | 21.6% → 25%+ | 낮음 |
| 6 | exit_click을 Supabase에도 트래킹 | 이탈 분석 가능 | 중간 |
| 7 | 유형 미리보기 티저를 히어로에 배치 | 호기심 유발 | 중간 |
